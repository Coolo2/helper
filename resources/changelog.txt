0.4.4 - new commands and improvements to existing ones.

__New commands:__

`{prefix}ascii` - Convert between ascii and text (works both ways)
`{prefix}morse` - Convert between morse and text (works both ways)

__Improvements:__

Added "Jump To Message" for message edit in logging
Rewritten `{prefix}binary` with a library I wrote, it now works both ways thanks to this
`{prefix}nick` now supports an all argument (saves previous nicknames for when you want to reset)

__Bug fixes:__

Fixed an issue with `{prefix}userinfo` when you aren't playing a game

__Website changes:__

Discord embed for the website

__Development changes:__

-The rewriting of some commands has lowered some file sizes
